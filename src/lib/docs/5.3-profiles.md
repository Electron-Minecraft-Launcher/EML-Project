# Profiles

You can use Profiles to fetch the list of profiles from the EML AdminTool website and launch Minecraft with a specific profile.

> [!WARNING]
> Profiles requires an [EML AdminTool](/docs/install-eml-admintool) website to work.

```typescript
import { Profiles } from 'eml-lib'

launcher()

async function launcher() {
  const profiles = new Profiles('https://at.emlproject.com')

  try {
    const profileList = await profiles.getProfiles()
  } catch (err) {
    console.error(err)
  }
}
```

### `Profiles` constructor

| Parameter | Type     | Description                            | Required? |
| --------- | -------- | -------------------------------------- | --------- |
| `url`     | `string` | The URL of your EML AdminTool website. | Yes       |

### `Profiles.getProfiles()` method

Fetch the list of profiles from the EML AdminTool website.

**Returns:** `Promise<IProfile[]>` - The list of Profile objects.

**Throws:** `FETCH_ERROR` - If a request to get data fails.
