Today, I'm announcing the **release of EML AdminTool and EML Lib** as stable 2.0.0 versions. This major milestone brings a completely reworked, faster update system for EML AdminTool, and introduces full ESM support alongside Yggdrasil authentication for EML Lib.

> ⚠️ EML AdminTool: Important notice for existing users
> If you are upgrading from a previous beta version, a clean installation is mandatory. Attempting to upgrade directly will break your setup. Please read the release notes [from GitHub](https://github.com/Electron-Minecraft-Launcher/EML-AdminTool/releases/tag/v2.0.0) for the exact step-by-step instructions to wipe the old containers and start fresh.

You can read the detailed lists of bug fixes and technical changes in the GitHub release notes:
- [EML AdminTool v2.0.0 Release Notes](https://github.com/Electron-Minecraft-Launcher/EML-AdminTool/releases/tag/v2.0.0)
- [EML Lib v2.0.0 Release Notes](https://github.com/Electron-Minecraft-Launcher/EML-Lib/releases/tag/v2.0.0)
