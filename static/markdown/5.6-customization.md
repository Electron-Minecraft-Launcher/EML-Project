# Customization

## News feed

You can use News to get the news from your EML AdminTool website and display them in your Launcher.

> [!WARNING]
> Bootstraps requires an [EML AdminTool](https://github.com/Electron-Minecraft-Launcher/EML-AdminTool-v2/) website to work.

<!-- CODESWITCH:START -->

<!-- CODE:JavaScript -->

```javascript
const { News } = require('eml-lib')

news()

async function news() {
  const news = new News('https://at.emlproject.qzz.io')

  try {
    const newsList = await news.getNews()
  } catch (err) {
    console.error(err)
  }
}
```


<!-- CODESWITCH:END -->

### `News` constructor

| Parameter | Type     | Description                            | Required? |
| --------- | -------- | -------------------------------------- | --------- |
| `url`     | `string` | The URL of your EML AdminTool website. | Yes       |

### `News.getNews()` method

Get all the news from the EML AdminTool.

**Returns:** `Promise<INews[]>` - The [news](https://github.com/Electron-Minecraft-Launcher/EML-Lib-v2/blob/main/types/news.d.ts).

**Throws:** `FETCH_ERROR` - If the request to the EML AdminTool website fails.

### `News.getCategories()` method

Get all the News categories from the EML AdminTool.

**Returns:** `Promise<INewsCategory[]>` - The [news categories](https://github.com/Electron-Minecraft-Launcher/EML-Lib-v2/blob/main/types/news.d.ts).

**Throws:** `FETCH_ERROR` - If the request to the EML AdminTool website fails.

### ~~`News.getNewsByCategory()`~~ method

> [!WARNING]
> This method is currently deprecated, and returns an empty array. 
> The EML AdminTool API does not support `News.getNewsByCategory()` anymore, but may be used in the future. Please use `News.getNews().filter(...)` instead.

## Background customization

You can use Background to get the background from your EML AdminTool website and display it in your Launcher.

> [!WARNING]
> Background requires an [EML AdminTool](https://github.com/Electron-Minecraft-Launcher/EML-AdminTool-v2/) website to work.

<!-- CODESWITCH:START -->

<!-- CODE:JavaScript -->

```javascript
const { Background } = require('eml-lib')

background()

async function background() {
  const background = new Background('https://at.emlproject.qzz.io')

  try {
    const currentBackground = await background.getBackground()
  } catch (err) {
    console.error(err)
  }
}
```
